# Paquete alineado (index + styles + app + idiomas)

Contenido:
- index.html
- styles.css
- app.js
- es.json
- ca.json
- en.json
- fr.json
- i18n_validation.json

Ajustes incluidos:
- app.js soporta data-i18n-content además de text/alt/title/placeholder/aria-label.
- El selector de idioma y las opciones del desplegable también son traducibles.
- Las claves de los JSON están alineadas con el index.html incluido en este paquete.
- Validación incluida en i18n_validation.json.

Recomendación de despliegue:
1. Sustituir todos estos ficheros a la vez.
2. Forzar recarga del navegador (Ctrl+F5) o abrir en incógnito.
3. Si usas GitHub Pages / Vercel, espera al nuevo deploy.
